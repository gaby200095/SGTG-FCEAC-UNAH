# Sistema de Gestión de Trámites de Graduación (SGTG) – FCEAC UNAH

Este es el frontend del proyecto "Sistema de Gestión de Trámites de Graduación (SGTG)" desarrollado con React.js y Tailwind CSS. Este sistema está diseñado para facilitar la gestión de trámites de graduación en la Facultad de Ciencias Económicas, Administrativas y Contables de la Universidad Nacional Autónoma de Honduras (UNAH).

## Requisitos

- Node.js (versión 14 o superior)
- npm (versión 6 o superior)

## Instalación

1. Clona el repositorio:

   ```
   git clone https://github.com/tu_usuario/SGTG-FCEAC-UNAH.git
   ```

2. Navega al directorio del frontend:

   ```
   cd SGTG-FCEAC-UNAH/frontend
   ```

3. Instala las dependencias:

   ```
   npm install
   ```

## Uso

Para iniciar la aplicación en modo de desarrollo, ejecuta:

```
npm start
```

Esto abrirá la aplicación en `http://localhost:3000`.

## Estructura del Proyecto

- **public/index.html**: Plantilla HTML principal.
- **src/assets**: Recursos estáticos como imágenes y fuentes.
- **src/components/Navbar.jsx**: Componente de la barra de navegación.
- **src/pages/Home.jsx**: Página principal de la aplicación.
- **src/App.jsx**: Componente principal que configura las rutas.
- **src/index.js**: Punto de entrada de la aplicación React.
- **src/styles/tailwind.css**: Estilos de Tailwind CSS.
- **tailwind.config.js**: Configuración de Tailwind CSS.
